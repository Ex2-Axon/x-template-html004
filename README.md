## คำอธิบาย

ยินดีต้อนรับสู่ **Lunaria** - มหานครแห่งราศีกรกฎ (The City of Cancer) 
หนึ่งในโปรเจ็กต์ภายใต้ Axon/html5 ที่ออกแบบมาเพื่อสะท้อนจิตวิญญาณแห่งธาตุน้ำ อารมณ์ที่ลึกซึ้ง และความใส่ใจในแบบฉบับของชาวราศีกรกฎื่อมโยงเนื้อหาในรูปแบบ interactive:

- หน้าแรกต้อนรับผู้เยี่ยมชม พร้อมภาพประกอบดวงจันทร์และฟีเจอร์ปรับอารมณ์เมือง
- แสดงแผนที่เมืองลูนาเรียพร้อมแลนด์มาร์กเชิงจินตนาการ
- เครื่องพยากรณ์อารมณ์ด้วยการเลือกเฟสจันทร์
- คลังความทรงจำพร้อมฟอร์มส่งข้อความ
- ตัวตรวจความเข้ากันได้ของราศี

## โครงสร้าง

- `index.html` - หน้าเว็บหลัก
- `css/styles.css` - สไตล์ CSS แบบแยกไฟล์
- `js/main.js` - โค้ด JavaScript แบบแยกไฟล์
- `components/header.html` - ส่วน header / navigation
- `components/footer.html` - ส่วน footer

## เทคโนโลยี

- Tailwind CSS CDN
- Google Fonts (Sarabun, Mitr)
- Lucide icons
- Component loader ผ่าน `assets/load-components.js`
